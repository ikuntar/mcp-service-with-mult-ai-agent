## 🎯 核心概念

### AI-Agent
- **功能性Agent**：简单任务，快速响应
- **高级Agent**：复杂任务，工具调用

### 提示词工程
- **系统提示词**：定义角色
- **追加提示词**：追加指令
- **两种格式**：固定/模板

## ✅ 设计理念

**简单即美**
- 最少的组件
- 清晰的职责
- 易于理解

**生产就绪**
- 错误处理

## 📊 架构概览

```
用户任务
    ↓
[Agent选择] → 功能性 或 高级
    ↓
[模型推理] → 功能性模型 或 高级模型
    ↓
[工具调用] (仅高级)
    ↓
[记忆存储]
    ↓
返回结果
```
